# Copa filmes

## Tecnologias utilizadas

- .Net Core 3.1
- AutoMapper
- Asp.Net Core WebAPI
- .NET Core Native DI

### Testes
- xUnit
- MOQ
- Bogus

## Executandando aplicação localmente
`CopaFilmes\copa-filmes-back-end\CopaFilmes.Service.API`
`dotnet run`
O back end por padrão executa nas portas http 5000 e https 5001
